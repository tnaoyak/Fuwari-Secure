Installation & Setup

Method A: Homebrew

The easiest way to install and keep updated. Run this command in your Terminal:

$ brew install tnaoyak/tap/fuwari-secure

If you see a security warning on first launch, please run the quarantine removal command shown in the troubleshooting box below.

Method B: Manual DMG

  1. Download the latest DMG file from the CTA above.
  2. Double-click the DMG and drag Fuwari-Secure.app to your Applications folder.
  3. Right-click the app in Applications, select Open, and click Open again to bypass Gatekeeper.

Troubleshooting: "App is damaged" Warning

If macOS shows a warning saying "Fuwari-Secure is damaged and cannot be opened. You should move it to the Trash.", the app is not damaged. This is macOS's default behavior for downloaded self-signed apps.

To fix this, you can easily remove the macOS quarantine attribute via Terminal:

Terminal Command Local Bypass
$ xattr -cr /Applications/Fuwari-Secure.app
  1. Open the Terminal app on your Mac (via Spotlight: Cmd + Space and type "Terminal").
  2. Copy and paste the command above, then press Enter.
  3. Close the Terminal and launch Fuwari-Secure.app again. It will open perfectly!
