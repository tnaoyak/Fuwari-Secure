企業向けセキュア版

スクリーンショットを、
付箋のように画面に浮かせる。

Fuwari-Secureは、撮影したスクリーンショットを画面の最前面にピン留めし、資料作成や開発時の参考として自由に配置できるMac用アプリです。

最前面にピン留め

撮影した画像を常に最前面に表示。ブラウザやエディタを切り替えても隠れず、資料を見ながらの作業に最適です。

自由なコントロール

ドラッグでのサイズ・位置変更はもちろん、マウスホイールを回すだけで不透明度を直感的に調整できます。

コピー＆ローカル保存

浮かべた画像は、ワンクリックでクリップボードにコピーしてチャットに貼り付けたり、ローカルに保存したりできます。

手動インストール

スタンドアロンの DMG ファイルをダウンロードします。アプリケーションフォルダにドラッグして、初回起動時にGatekeeperをバイパスします。

Download .dmg (最新版)

Homebrew で導入

Homebrew経由で簡単にインストールとアップデートが行えます。開発者やコマンドラインユーザーに最適です。

$ brew install tnaoyak/tap/fuwari-secure
簡単なアップデートでおすすめ

完全オフライン版

企業の厳格なセキュリティ基準に適合するため、外部へのアップロード機能を完全に排除しています。

  • 完全オフライン動作（外部通信なし）
  • Gyazo等のクラウド送信機能なし
  • アカウント作成やOAuth連携不要
Fuwari-Secure デモ

*Fuwari-Secureは、Fuwariの直感的で優れた「画面に浮かせる」機能をすべて引き継いでいます。

インストールと起動手順

方法 A: Homebrew (推奨)

最も簡単にインストールとアップデート管理が行えます。ターミナルで以下のコマンドを実行してください：

$ brew install tnaoyak/tap/fuwari-secure

※ 初回起動時にセキュリティ警告が表示される場合は、下部のトラブルシューティングに記載されている隔離属性解除コマンドを実行してください。

方法 B: 手動 (DMG)

  1. 上部のダウンロードエリアから最新の DMG ファイルをダウンロードします。
  2. DMG をダブルクリックして開き、Fuwari-Secure.app を アプリケーション（Applications） フォルダにドラッグします。
  3. アプリケーションフォルダ内のアプリを右クリックして「開く」を選択し、Gatekeeper警告をバイパスして起動します。

トラブルシューティング：「壊れているため開けません」と表示される場合

ダウンロードしたアプリを起動しようとした際、macOSが「Fuwari-Secureは破損しているため開けません。ゴミ箱に入れる必要があります。」という警告を表示することがあります。これは、インターネットからダウンロードされた未署名アプリに対するmacOSの標準的なセキュリティ制限（隔離機能）によるもので、アプリは実際には破損していません。

この問題は、以下の手順で「ターミナル」から隔離フラグを解除することで、簡単に解決できます。

ターミナルに貼り付けるコマンド 隔離属性の解除
$ xattr -cr /Applications/Fuwari-Secure.app
  1. Macの「ターミナル」アプリを起動します（Spotlight検索 Cmd + Space で「ターミナル」または「Terminal」と入力して起動できます）。
  2. 上記のコマンド（xattr...）をコピーしてターミナルに貼り付け、Enter キーを押します。
  3. ターミナルを閉じ、再度 Fuwari-Secure.app を起動します。警告が消え、正常に起動できるようになります！
